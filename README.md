Patient Health Data Management and Analysis Tool

A web-based patient log and health analysis tool built using HTML and Python (Brython).
The system stores patient records and analyzes health parameters such as BMI and blood
pressure to provide basic health insights and recommendations.

Files
- patient_log_ui.html – main user interface
- report.pdf – project report
Technologies Used
- HTML
- Python (Brython)
How to Run
Open `patient_log_ui.html` in a browser that supports Brython.
